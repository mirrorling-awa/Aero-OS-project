# Aero-OS-project
感谢您使用Aero OS！
Aero OS是一个基于scratch 3.0的精简图形化操作系统
系统本体 by 【镜墨工作室】博丽镜泠
界面设计 by 【镜墨工作室】古明地镜灵
离线页面设计 by 【镜墨工作室】白镜泽
其他 by 所有镜墨工作室的成员们
特别鸣谢：神笔工作室
          星光工作室
          @共创世界 小a欸_


使用说明：由于scratch没有可供此作品接入互联网的插件，所以“泠泽浏览器”仅供预览，所有页面均为离线页面。另外，理论上可以通过共创世界编辑器独有的云插件实现互联网的部分功能。
此功能正在开发中，敬请期待。
